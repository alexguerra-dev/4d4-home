# 4d4.live Homepage

The [page](https://home.4d4.live) is a way to link to different project. Kind of like a portfolio piece.

Links to different projects

- [BetoBar](https://betobar.4d4.live)
- [Synth](https://synth.4d4.live)
- [Boop](https://boop.4d4.live)

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
